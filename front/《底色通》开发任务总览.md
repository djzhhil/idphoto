## 🧩 《底色通》开发任务总览

| 阶段       | 版本        | 目标                             | 开发周期 |
| ---------- | ----------- | -------------------------------- | -------- |
| 🚀 第一阶段 | V1.0 基础版 | 实现上传 → 换底 → 下载完整流程   | 3~5 天   |
| 🔧 第二阶段 | V1.1 增强版 | 优化体验、增加裁剪与美颜         | 1~2 周   |
| 📸 第三阶段 | V2.0 专业版 | 自动换装 + 背景自定义 + 批量处理 | 3~4 周   |
| 💰 第四阶段 | V3.0 商业版 | 增加会员体系、支付与分享裂变     | 1~2 月   |

------

## 🚀 第一阶段：V1.0 基础版（MVP）

### 🎨 前端（UniApp 小程序）

-  图片上传组件（相册 / 拍照）
-  Base64 转换与本地预览
-  选择背景颜色（白、红、蓝）
-  调用后端接口更换背景色
-  显示生成后的预览图
-  下载 / 保存到相册
-  简洁的 UI 布局 + 按钮状态禁用逻辑
-  错误提示（如文件过大、网络失败）

### ⚙️ 后端（Spring Boot / Java）

-  `/api/remove-bg` 接口：接收 base64 图像并返回处理后 base64
-  集成 remove.bg / 自研背景替换模型
-  支持白、红、蓝三种背景替换参数
-  部署到 Railway / Vercel / 服务器
-  跨域配置（CORS 支持）

------

## 🔧 第二阶段：V1.1 增强版（体验升级）

### 🎨 前端

-  新增「尺寸选择」组件（1寸、2寸、自定义）
-  添加「自动居中裁剪」功能（前端居中人像）
-  增加「磨皮 / 美颜」选项（前端滤镜）
-  添加加载动画 / 进度条
-  历史记录缓存（`uni.setStorageSync`）
-  优化错误提示与交互体验

### ⚙️ 后端

-  新增 `/api/resize` 接口（生成 1寸 / 2寸规格）
-  支持智能人脸检测 + 居中裁剪
-  集成 OpenCV 或第三方美颜 API
-  图片压缩 / 清晰度优化
-  优化性能（异步处理 + 限流）

------

## 📸 第三阶段：V2.0 专业版（功能拓展）

### 🎨 前端

-  增加「换装模板」选项（衬衫、西装）
-  背景自定义（输入 HEX 或调色盘）
-  批量生成多个底色
-  云端保存历史记录（需授权登录）
-  更专业的 UI（加入 uView / 自定义主题）
-  “智能推荐背景” 功能（根据服装 / 性别推荐）

### ⚙️ 后端

-  `/api/change-clothes`：服装替换接口（基于 AI 生成）
-  `/api/multi-bg`：支持一次生成多种底色
-  图片云存储（OSS / COS）
-  用户系统（openid 绑定）
-  数据库设计（用户、历史记录、订单）
-  接口鉴权（JWT / 微信登录态）

------

## 💰 第四阶段：V3.0 商业化版本

### 🎨 前端

-  增加会员中心页面（会员权益说明）
-  内购入口（支付开通会员）
-  广告插屏（激励观看解锁功能）
-  一键分享（好友助力）
-  生成带 Logo 的分享海报

### ⚙️ 后端

-  `/api/pay`：微信支付下单接口
-  `/api/vip/check`：会员状态查询
-  积分 / 次数系统（记录用户使用情况）
-  推广返利逻辑（分享助力统计）
-  日志与监控系统
-  异步任务队列（图片处理解耦）
