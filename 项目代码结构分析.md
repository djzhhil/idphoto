## 🔧 后端技术栈

### 核心框架
- **Spring Boot** 3.5.4
- **Java** 17
- **Maven** 构建工具

### 第三方服务
- **腾讯云 BDA API**（Body Database Analysis）- 人像分割/抠图
- **版本：** tencentcloud-sdk-java 3.1.713

### 图像处理
- **Java AWT** - 原生图像处理库
- **BufferedImage** - 图像缓冲
- **Graphics2D** - 2D绘图
- **ImageIO** - 图像IO

### 配置要求
```yaml
# application.properties
tencent.secret-id=你的腾讯云SecretId
tencent.secret-key=你的腾讯云SecretKey
tencent.region=ap-shanghai
```

## 🌐 API 接口说明

### 后端接口（3个）

#### 1. **POST /api/remove-bg** - 去底色（旧接口）
```
请求：
{
  "image": "data:image/jpeg;base64,...",
  "background": "white/red/blue"
}

响应：
{
  "image": "data:image/jpeg;base64,...",
  "message": "生成成功"
}
```

#### 2. **POST /api/preview-idphoto** - 预览证件照（新接口）
```
请求：
{
  "image": "data:image/jpeg;base64,...",
  "size": "1inch/2inch",
  "brightness": 50,
  "smoothness": 5,
  "bgColor": "white/red/blue",
  "previewMode": true,
  "quality": "quick"
}

响应：
{
  "image": "data:image/jpeg;base64,...",
  "message": "预览生成成功(缓存)"
}
```

#### 3. **POST /api/generate-idphoto** - 生成证件照（高质量）
```
请求：
{
  "image": "data:image/jpeg;base64,...",
  "size": "1inch/2inch",
  "brightness": 50,
  "smoothness": 5,
  "bgColor": "white/red/blue"
}

响应：
{
  "image": "data:image/jpeg;base64,...",
  "message": "生成成功"
}
```

## 🎨 功能模块分析

### ✅ 已实现功能

#### 后端功能（ImageProcessingService）

1. **图片尺寸调整**
   - 智能裁剪（保持人像居中）
   - 标准证件照尺寸：1寸(295x413)、2寸(413x579)
   - 支持小一寸(260x378)

2. **亮度调整**
   - 范围：0-100%
   - 使用 RescaleOp 实现亮度调整

3. **磨皮处理**
   - 范围：0-20
   - 使用 ConvolveOp 实现模糊滤镜效果

4. **背景色替换**
   - 支持颜色：白色、红色、蓝色、绿色、灰色
   - 使用 Graphics2D 绘制背景

5. **图片缓存**
   - 基于内存缓存（最大50张）
   - 缓存键：基于图片内容+尺寸+参数生成
   - 自动清理旧缓存

6. **内存优化**
   - OutOfMemoryError 处理
   - 自动GC触发
   - 支持低质量/高质量两种输出模式

7. **腾讯云抠图集成**
   - 调用人像分割 API
   - Base64 透明背景返回
   - 异常处理和错误提示

#### 前端功能

##### index.vue（首页）
- 照片上传（相册/拍照）
- 图片压缩（减少内存占用）
- 图片尺寸验证（最小200x200）
- Base64 转换
- 加载状态提示

##### edit.vue（编辑页）
- 实时预览（通过后端API）
- 背景颜色选择（5种颜色）
- 尺寸选择（1寸、2寸、小一寸）
- 亮度调整（Slider，0-100%）
- 磨皮调整（Slider，0-20）
- 处理状态指示
- 预览缓存

##### result.vue（结果页）
- 图片展示
- 保存到相册
- 重新制作
- 保存成功提示
- 分享提示

##### request.js（请求工具）
- 环境配置（本地/生产）
- GET/POST 封装
- 统一错误处理
- CORS 配置

## 🔍 代码质量分析

### ✅ 优点
1. **代码结构清晰** - 按模块分层（controller/service/dto/config）
2. **功能完善** - 涵盖证件照制作的所有核心功能
3. **缓存机制** - 减少重复计算
4. **错误处理** - 有完整的异常捕获和用户提示
5. **内存优化** - 处理大图时的内存保护
6. **异步处理支持** - 预览模式和高质量生成分开

### ⚠️ 需要改进的地方

#### 后端改进
1. **配置文件缺失**
   - `application.properties` 文件不存在
   - 需要添加腾讯云配置示例

2. **缺少日志记录**
   - 没有 Slf4j 日志框架
   - 错误信息只是打印到控制台

3. **缺少单元测试**
   - 没有 test 目录
   - 建议添加 Service 和 Controller 测试

4. **硬编码颜色值**
   - `#1890ff` 等颜色值应该提取为常量

5. **API 文档缺失**
   - 没有 Swagger/OpenAPI 文档
   - 建议添加 Swagger 注解

#### 前端改进
1. **缺少配置管理**
   - 环境配置硬编码在 request.js
   - 建议使用 config 目录分离配置

2. **错误码统一**
   - 前后端错误码不统一
   - 建议定义标准错误码枚举

3. **缺少加载动画**
   - 只有文字提示
   - 建议添加 loading 组件

4. **图片压缩优化**
   - 前端只使用 compressed 模式
   - 可以根据图片大小动态调整压缩比例

5. **用户引导缺失**
   - 没有新手引导或帮助页面
